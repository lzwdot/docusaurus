# Vue3 对 Vue2 有优势更快

更快就是优势

## Vue3 的优势

- 性能更好
- 体积更小
- 更好的 ts 支持
- 更好的代码组织
- 更好的逻辑抽离
- 更多的新功能

## 为何 Vue3 对 Vue2 快

- Proxy 响应式
- PatchFlag
- hoistStatic
- cacheHandler
- SSR 优化
- tree-shaking
