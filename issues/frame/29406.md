# React 性能优化

- 渲染列表是加 key
- 自定义事件，DOM 事件及时销毁
- 合理使用异步组件
- 减少函数 bind this 的次数
- 合理使用 SCU PureComponent 和 memo
- 合理使用 immutable.js
- webpack 层面的优化
- 前面通用的性能优化，如图片懒加载
- 使用 SSR
