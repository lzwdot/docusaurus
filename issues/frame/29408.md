# React 和 Vue 的区别

共同点

- 都支持组件化
- 都是数据驱动视图
- 都使用 vdom 操作 DOM
  
区别

- React 使用 JSX 拥抱 JS，Vue 使用模板拥抱 html
- React 函数式编程: `this.setState({a:100})`，Vue 声明式编程: `this.data.a = 100`
- React 更新需要自力更生，Vue 把想要的都给你
