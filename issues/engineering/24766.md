# Git 强制更新覆盖本地

当需要强制使用远程分支覆盖本地分支，可以使用以下几行命令，比如：强制更新为远程 `master` 分支

``` js 
git fetch --all
git reset --hard origin/master
```

`git fetch` 只是下载远程的库的内容，不做任何的合并； `git reset` 把 `HEAD` 指向刚刚下载的最新的版本
