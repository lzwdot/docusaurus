# 常用的 git 命令

常用的 git 命令

``` js 
// 克隆代码库
git clone  git@github.com:lzwdot/git-demo.git
// 查看文件修改状态
git status
// 查看分支
git branch
// 查看修改内容
git diff [filename]
// 提交文件
git add [.|filename]
// 生成提交记录
git commit -m "初次修改"
// 查看提交日志
git log
// 查看当前提交内容，[hash]为 git log 中的日志 id
git show [hash]
// 配置信息
git config user.name lzw
git config user.email lzw22@qq.com
// 撤销所有或单个文件修改
git checkout [.|filename]
// 上传 master 分支代码到服务器
git push origin master
// 下载 master 分支最新代码
git pull origin master
// 新建分支
git checkout -b [branch]
// 切换 master 分支
git checkout master
// 拉取服务器所有分支
git fetch
// 合并分支到 master
git merge [branch]
// 当前修改暂时缓存，[放出暂时缓存文件]
git stash [pop]
// 使用 https clone 时，配置记住密码
`git config --global` `credential.helper store`
```
