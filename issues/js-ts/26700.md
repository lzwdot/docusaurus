# 数组去重，考虑计算效率

数组去重的方法

- 传统方式，遍历元素挨个比较，去重

``` js 
// 传统方式
function unique(arr) {
    const res = []
    arr.forEach(item => {
        if (res.indexOf(item) < 0) {
            res.push(item)
        }
    })
    return res
}
console.log(unique([1,1,2,2])) // [1,2]
```

- 使用 ES6 中的 Set，性能更好推荐使用

``` js 
// 使用 Set （无序，不能重复）
function uniqueSet(arr) {
    const res = new Set(arr)
    return [...res]
}
console.log(uniqueSet([1,1,2,2])) // [1,2]
```
