# 什么是 JSON ?

JSON

- json 是一种数据格式，本质是一段字符串
- json 格式和 JS 对象结构一致，对 JS 语言更友好
- window.JSON 是一个全局对象：JSON.stringify JSON.parse

``` js 
// key 和字符必须使用双引号
const json = {
    "name": "张三",
    "age": 19
}
```
